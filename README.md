# Family Trivia

Juego de preguntas familiar independiente.

## Entradas

- `index.html` - juego principal.
- `ruletas.html` - ruletas para formar equipos antes de empezar.

## Uso

Abre `index.html` directamente en el navegador o despliega esta carpeta como proyecto estático.

La carpeta mantiene sus propias copias de `css/`, `js/`, `img/` y `audios/` para funcionar sin depender del proyecto original.
